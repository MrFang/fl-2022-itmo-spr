Cо стоковым состоянием
|   | A | B | C | D | E | _
| - | - | - | - | - | - | -
| A | V | - | - | - | - | -
| B | x | V | - | - | - | -
| C | x | x | V | - | - | -
| D | V | x | x | V | - | -
| E | x | x | x | x | V | -
| _ | x | x | x | x | x | V

```mermaid
flowchart LR
    AD -- 0 --> B[[B]]
    AD -- 1 --> C[[C]]
    B -- 0 --> _
    B -- 1 --> AD
    C -- 0 --> _
    C -- 1 --> E
    E -- 0 --> C[[C]]
    E -- 1 --> _
    _ -- 0 --> _
    _ -- 1 --> _
```

Без стокового состояния

|   | A | B | C | D | E
| - | - | - | - | - | -
| A | V | - | - | - | -
| B | x | V | - | - | -
| C | x | x | V | - | -
| D | V | x | x | V | -
| E | x | x | x | x | V

```mermaid
flowchart LR
    AD -- 0 --> B[[B]]
    AD -- 1 --> C[[C]]
    B -- 1 --> AD
    C -- 1 --> E
    E -- 0 --> C[[C]]
```
